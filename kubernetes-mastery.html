Week-by-Week Curriculum

Deep dive into the control plane, nodes, and core objects like Pods, Deployments, and Services.

Advanced deployment strategies, service discovery, and exposing applications with Ingress.

Managing stateful applications with PersistentVolumes and externalizing configuration with ConfigMaps and Secrets.

Package management with Helm, monitoring with Prometheus, and securing clusters with RBAC.

Frequently Asked Questions

While we will have examples on AWS EKS, the majority of the course can be completed on your local machine using tools like Minikube or Kind. We will guide you through the setup for both local and cloud environments.

This course focuses on practical, real-world application of Kubernetes in a DevOps workflow, including CI/CD, monitoring, and package management. While it covers many CKA/CKAD topics, its goal is job-readiness rather than pure exam preparation.

We will not be building a production cluster from the ground up (e.g., "Kubernetes the Hard Way"), as this is an advanced topic. We will focus on using managed Kubernetes services (like EKS) and local clusters (like Minikube) to deploy and manage applications, which is the more common real-world scenario.
