Docker Administration

Learn to build, manage, and secure Docker containers in production

What You'll Learn

Build & Manage Docker Images

Create efficient and secure container images using Dockerfile best practices.

Configure Docker Networking

Understand and implement various Docker networking models for container communication.

Handle Persistent Data

Manage application data using Docker volumes and bind mounts for stateful services.

Secure Docker Environments

Learn best practices for securing the Docker daemon and containers in production.

Course Details

Duration: 4 Weeks

Mode: Online / Hands-on Labs

Last Updated: Nov 2023

Level: Intermediate

Prerequisites: Linux & Networking Basics

Who is this course for?

This course is ideal for developers and sysadmins looking to master containerization.

Software Developers

Developers who want to package their applications for consistent deployments.

System Administrators

Sysadmins aiming to modernize application deployment and management.

QA Engineers

QA professionals looking to create isolated and reproducible testing environments.

Week-by-Week Curriculum

Understand Docker architecture, manage containers and images, and write multi-stage Dockerfiles.

Explore Docker networking models and learn to manage persistent data with volumes.

Define and run multi-container applications for local development and testing.

Learn to scan images for vulnerabilities and apply security best practices for production environments.
