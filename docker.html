Docker Administration

Learn to build, manage, and secure Docker containers in production

Course Overview

This Docker Administration course focuses on real-world container management skills. You will learn how Docker works internally, manage containers and images, configure networking and storage, and follow production best practices.

Course Outline

  • Docker Architecture & Components
  • Installing Docker on Linux
  • Images, Containers & Dockerfile
  • Docker Registry & Image Management
  • Docker Networking (Bridge, Host, Overlay)
  • Volumes & Persistent Storage
  • Docker Compose
  • Monitoring & Logs
  • Docker Security Best Practices
  • Real-World Production Scenarios
Course Details

Duration: 4 Weeks

Level: Intermediate

Prerequisites: Linux & Networking Basics
