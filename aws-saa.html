Who is this course for?

This course is for anyone aiming to design robust and efficient cloud solutions.

Aspiring Solutions Architects

Individuals aiming for a career in cloud architecture and design.

Developers

Developers who want to design and build well-architected applications on AWS.

IT Professionals

Sysadmins and IT staff looking to transition into a cloud-focused role.

Course Curriculum

  • What is Cloud Computing
  • Cloud Deployment Models (Public, Private, Hybrid)
  • Cloud Service Models (IaaS, PaaS, SaaS)
  • AWS Global Infrastructure (Regions, AZs, Edge Locations)
  • Shared Responsibility Model
  • AWS Management Console, CLI, and SDKs

  • IAM Users, Groups, Roles
  • IAM Policies & Principle of Least Privilege
  • MFA and Security Best Practices
  • Cross Account Access & IAM Roles for Services

  • Amazon VPC fundamentals, CIDR, and IP addressing
  • Subnets (Public / Private) & Route Tables
  • Internet Gateway & NAT Gateway
  • Network ACL vs Security Groups
  • VPC Peering, VPC Endpoints, Transit Gateway
  • Hybrid connectivity (VPN / Direct Connect)

  • EC2 instance types & pricing models
  • Auto Scaling, Launch Templates, Placement Groups
  • Elastic Load Balancer (ALB, NLB, Gateway LB)

  • Amazon S3 (Storage classes, Lifecycle, Versioning, Replication)
  • EBS (Volume types, Snapshots)
  • EFS, FSx, and Storage Gateway

  • Amazon RDS (Multi-AZ, Read Replicas)
  • Amazon Aurora & Amazon DynamoDB
  • Amazon ElastiCache
  • Database migration strategies (DMS)

  • Multi-AZ architecture design
  • Auto Scaling and Load balancing design patterns
  • Disaster recovery strategies (Backup/Restore, Pilot Light, Warm Standby, Multi-Site)

  • AWS Lambda and event-driven architecture
  • API Gateway & Step Functions
  • SQS and SNS for decoupling applications

  • Amazon CloudWatch (Metrics, Alarms, Logs)
  • AWS CloudTrail for auditing
  • AWS Config & AWS X-Ray

  • AWS Shield & AWS WAF
  • AWS KMS & Secrets Manager
  • Encryption best practices in AWS

  • AWS Pricing models & Cost Explorer
  • Savings Plans & Trusted Advisor
  • Cost optimization strategies and tagging

  • 3-tier, Microservices, Serverless, and Hybrid architectures
  • AWS Well-Architected Framework pillars

  • AWS Migration Hub & Application Migration Service (MGN)
  • Database Migration Service (DMS)
  • AWS Snowball family for large data transfers

Frequently Asked Questions

This course is specifically designed to cover all domains of the SAA-C03 exam blueprint. While we provide the knowledge and hands-on experience, we always recommend additional practice exams to ensure exam readiness.

We make extensive use of the AWS Free Tier. We will guide you on how to stay within the free limits to minimize costs. Any services outside the Free Tier will be clearly communicated.

The Solutions Architect course focuses on the 'what' and 'why' of designing AWS architectures (e.g., which service to use for a given requirement). The AWS DevOps course focuses on the 'how' of implementing, automating, and operating those architectures using CI/CD pipelines and DevOps practices.
