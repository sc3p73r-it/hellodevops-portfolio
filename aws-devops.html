AWS DevOps

Implement CI/CD, automation, and infrastructure as code on the AWS cloud.

Course Overview

This course is designed for professionals looking to master DevOps practices on AWS. You will learn to use AWS services to build, deploy, and manage scalable and resilient applications.

Course Outline

  • Introduction to DevOps on AWS
  • IAM & AWS Security Best Practices
  • VPC, EC2, and Core Infrastructure
  • Infrastructure as Code with CloudFormation & Terraform
  • CI/CD with AWS CodePipeline, CodeBuild, CodeDeploy
  • Containerization with ECS & ECR
  • Introduction to Kubernetes on AWS (EKS)
  • Monitoring with CloudWatch & Logging with CloudTrail
  • Final Project: Deploying a full application on AWS
Course Details

Duration: 8 Weeks

Mode: Online / Live Sessions

Level: Intermediate to Advanced

Prerequisites: AWS Basics, Linux, CI/CD Concepts
