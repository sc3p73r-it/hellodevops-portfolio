Linux System Administration

Build strong Linux skills required for DevOps and Cloud roles.

Course Overview

This course focuses on practical Linux administration skills used in production environments. You will gain hands-on experience managing users, services, storage, networking, and security.

Course Outline

  • Linux Architecture & Distributions
  • File System & Permissions
  • User & Group Management
  • Package Management (yum/dnf/apt)
  • Process & Service Management (systemd)
  • Disk, LVM & Mounting
  • Networking & Firewall (firewalld)
  • Shell Scripting Basics
  • Security Hardening & Logs
  • Real-World Admin Scenarios
Course Details

Duration: 5 Weeks

Mode: Online / Hands-on Labs

Level: Beginner to Intermediate

Prerequisites: Basic computer knowledge
