Linux System Administration

Build strong Linux skills required for DevOps and Cloud roles.

What You'll Learn

Command Line Proficiency

Become confident and efficient navigating and managing a Linux system from the terminal.

Manage Users & Permissions

Learn to create users, manage groups, and control file access using standard permissions.

Configure Networking & Firewalls

Set up network interfaces and secure your system by managing firewall rules.

Automate Tasks with Shell Scripting

Write Bash scripts to automate repetitive system administration tasks.

Who is this course for?

This course is for anyone who needs to work with Linux servers, which is a must for cloud and DevOps.

Aspiring Cloud/DevOps Engineers

Linux is the foundation of the cloud. This course is the perfect starting point.

IT Support Staff

Level up your skills to manage servers and progress in your system administration career.

Anyone new to Linux

If you're curious about Linux and want a structured, hands-on learning path.

Week-by-Week Curriculum

Navigating the Linux filesystem and mastering file permissions and ownership.

Creating and managing users/groups, and installing/updating software with package managers.

Managing system services with systemd and configuring network interfaces and firewalls.

Working with disks and LVM, writing basic shell scripts, and applying security best practices.
